# 🏎️ Simple Racing Game

A simple **2D browser-based racing game** built using **HTML, CSS, and JavaScript**.
The player controls a car and must avoid enemy cars coming from the top of the road.
The score increases as long as the player survives.

---

## 🎮 Game Features

* 🚗 Player-controlled car
* 🚘 Random enemy cars
* ⌨️ Keyboard controls
* 💯 Score tracking
* 💥 Collision detection
* 🔁 Automatic game restart on crash

---

## 🛠️ Technologies Used

* **HTML5** – Game structure
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Game logic and interactions

---

## 🎯 Controls

| Key           | Action         |
| ------------- | -------------- |
| ⬅ Arrow Left  | Move car left  |
| ➡ Arrow Right | Move car right |

Avoid hitting the enemy cars to keep increasing your score.

---

## 🧠 Game Logic

1. The player car is positioned at the bottom of the screen.
2. Enemy cars spawn randomly at the top.
3. Enemy cars move downward continuously.
4. If an enemy collides with the player:

   * The game ends
   * Score is displayed
5. If the enemy passes without collision:

   * Score increases.
